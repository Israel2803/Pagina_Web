Listas

Primero se definen y dentro se les colocan un < li>< /li> para hacer los numerales

De estas hay distintos tipos:

Listas ordenadas < ol>< /ol>

  1. Pedro
  2. Lourdes
  3. José

Listas desordenadas < ul>< /ul>

  • Ayer
  • Hoy
  • Mañana

Listas de Definición < dl>< /dl>

Para las de Definición es diferente porque para la palabra es < dt>< /dt> y para la definición es < dd>< /dd>

Tablas

Las tablas eran antes el metodo para ordenar todo en una página pero ahora todo se hace en Css. La estructura de una tabla es la siguiente:

< table> < tr> < td> < /tr> < /table>

Para que se junten dos secciones de una tabla en horizontal se usa un colspan como atributo y en vertical se usa un rowspan, dependiendo de cuantas secciones se quieran juntar se le da valor al atributo
