Cotroles Básicos para una Página Web

Como primer punto debemos saber acerca de como se comienza una página. Esto es para que la pueda identificar cualquier navegador en cualquier parte:

La cabecera

< !DOCTYPE html>
< html lang="es">
< head>
< meta charset="utf-8">
< title>Título
< /head>
< body>

El final

< /body>
< /html>

El doctype es el encargado de definir el documento como Html5.
El html es el que encierra todo el código.
El head es información de la página para que el navegador realize ciertas operaciones, ademas esta parte no se ve.
El meta determina que los caracteres que se emplearan incluyen todos los idiomas.
El title determina el título que recibe la página.
El body es la parte en donde se escribe el código que se puede ver.

Enlaces

Los enlaces en una página web se utilizan para que la página no sea muy extensa y para llevarte a otro sitio al que el diseñador de la página te quiere mandar. Estos enlaces pueden ser en su misma página, en su misma carpeta o a páginas de terceros.

Para ellos se utilizan la etiqueta < a href="dirección aqui">< /a>
